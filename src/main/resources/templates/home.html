Load Generator

Creating different types of application load since 2021

Choose load type:

Available load generators:

Parameters:

  • Load percentage

Description:

Steady CPU load equal to given load percentage

Parameters:

  • Starting load percentage
  • Indecisiveness - load percentage change
  • Change speed - how many seconds pass between load percentage changes

Description:

CPU load starts at given load percentage, then jump up by given step amount. Speed parameter decides how often (aprox. in seconds) jump is made. When load reaches 100% step value is reversed, and load goes back to starting load percentage. Then cycle repeats indefinitely.

Parameters:

  • Memory load in megabytes

Description:

Load generator reserves given amount of megabytes of system memory. After finishing load test system tries to initialize garbage collection
